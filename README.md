 🤖 AI Recruitment System

An AI-based recruitment system that helps recruiters find the best candidates by analyzing resumes and matching them with job requirements.
The system automates the hiring process and reduces manual effort in candidate screening.

---

 📌 Problem Statement

Traditional recruitment processes require manual screening of hundreds of resumes, which is time-consuming and inefficient.
This project uses Artificial Intelligence to automatically analyze resumes and shortlist the most suitable candidates for a job.

---

 🎯 Objectives

* Automate resume screening process
* Reduce recruiter workload
* Improve candidate-job matching
* Speed up the hiring process

---

 🚀 Features

* Resume upload and analysis
* AI-based candidate ranking
* Job description matching
* Candidate recommendation
* Recruiter dashboard
